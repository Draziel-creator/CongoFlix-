Note : Guide d'utilisation

Bienvenue sur la vitrine du cinéma congolais ! Pour profiter de votre série dans les meilleures conditions, merci de suivre ces quelques étapes simples :

1. Avant de lancer la vidéo : Assurez-vous que la "Rotation automatique" de votre téléphone est activée dans votre menu de raccourcis Android (icône avec un petit téléphone qui tourne).

2. Rotation de l'écran : Une fois activée, tournez votre téléphone à l'horizontale. Le bouton de lecture deviendra plus facile à utiliser.

3. Agrandir l'image : Pour passer en plein écran, maintenez votre doigt appuyé une seconde sur le petit carré en bas à droite du lecteur, puis relâchez.

4. Gérer la publicité : Si une barre publicitaire apparaît, cliquez sur "Close Ad" (bouton noir juste au-dessus) pour la fermer.

5. Téléchargement : Pour enregistrer l'épisode, cliquez sur la petite flèche de téléchargement située directement à l'intérieur du lecteur.

Note importante : Si une nouvelle page s'ouvre par erreur, fermez-la pour revenir sur l'application.

Bon visionnage sur CongoFlix !
